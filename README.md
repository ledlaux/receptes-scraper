# receptes-scraper
Latvian cooking recipe scrapers using python 3.7 and Beautifulsoup.

With this scripts you can parse recipes from:
gardedis.lv, receptes.lv and tasty.lv.

Made for educational purposes only. This code is used for creation of telegram bot named Virtuves_bot, which helps to find cooking recipes. 
Try out bot at: https://telegram.me/virtuves_bot.
